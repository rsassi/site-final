HiveMind JMX

The HiveMind JMX library (module hivemind.jmx) brings the benefits of mature application management and monitoring to your HiveMind based applications by integration of the Java Management Extension (JMX).

With minimal effort you get access to these features:

  • Change log levels in your running application via a web based interface or by use of a management console
  • Measure the performance of middleware calls and view the data online
  • Make your services configurable at runtime. Change database urls or cache sizes
The services of this library are implemented in the module hivemind.management. The module is defined by a xml module descriptor.
