HiveMind 2

HiveMind 2 is on its way.

Its main features are:

  • XML dependencies removed from core and placed in separate hivemind-xml library
  • Pure java API for the definition of modules (services, configurations etc.)
  • Annotation based API for the definition of modules
  • Explicit and extensible Autowiring Service
  • Configurations not longer restricted to lists and maps
Expect to find some info here soon and an alpha version in February 2007.
