General

Keep up to date with the latest events in the HiveMind world.

First alpha release of HiveMind 2

A public alpha version of HiveMind 2 is available. It is available via the Maven repository or from the HiveMind download page. First documentation is available on the new wiki.

New Site

The HiveMind site has experienced a major redesign as preparation for the oncoming first release of HiveMind 2

HiveMind - Now Top Level

HiveMind is now a top-level Apache project. The new home page is http://hivemind.apache.org.
